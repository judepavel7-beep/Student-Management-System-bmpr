# Student Management System

This project consists of designing the complete architecture of a university student management application.

The goal of this project is to model and organize how a system can manage student information within a university environment.

## Project Objective

The objective of this project is to design the structure of an application that allows:

- Student registration
- Storage of student information
- Management of student data
- Basic operations on student records

This project focuses mainly on the architecture and organization of the system.

## Technologies Used

- Programming concepts
- Database design
- System architecture
- UML modeling

## Project Structure

The project is organized around different components that represent the structure of the application and how the system manages student data.

## Learning Purpose

This project was developed as part of academic training in computer science. It aims to understand how to design and organize the architecture of a software application used in a university environment.
